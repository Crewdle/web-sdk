Crewdle Samples

Explore our collection of interactive samples. You can also access our guides, API references and GitHub repository to learn more about the Crewdle Mist Computing Platform.

Chat Sample

Play around with a chat app sample that uses the Crewdle Key Value Database to store messages and users as well as the Crewdle Object Store to store files.

Monitoring Console Sample

Visualize the Crewdle Pub/Sub Topic in use with a monitoring console sample that displays CPU, Memory, and Disk usage. Each component is simulated as a node in the cluster transfering data through the Pub/Sub.

Guides Sample

Access the smaller samples to learn more about the Crewdle Mist Computing Platform and its underlying services.
